关于编程的几点一般建议：如果想成为一名专业的程序员，需要掌握的不仅仅是语言的语法，具体来说，应该了解你的工具，了解你的库，并了解你的文档。
与C编译相关的工具是gcc、gdb和ld。还有大量的库函数也可供你使用，但幸运的是libc包含了许多功能，默认情况下它与所有C程序相关联——需要做的就
是包含正确的头文件。最后，了解如何找到所需的库函数（例如，学习查找和阅读手册页）是一项值得掌握的技能。
